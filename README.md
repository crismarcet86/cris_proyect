# Cris Proyect - Backend con Node.js y MySQL

Este es un backend desarrollado en Node.js utilizando Express para la API REST y Sequelize como ORM para la conexión con una base de datos MySQL.

## 🛠️ Arquitectura del Proyecto

El proyecto sigue una estructura limpia basada en capas:
- `src/config`: Configuraciones del sistema y conexión a la base de datos.
- `src/models`: Definición de los modelos de Sequelize (Tablas de MySQL).
- `src/controllers`: Lógica de control de peticiones HTTP (Request/Response).
- `src/routes`: Definición de los endpoints de la API.
- `src/services`: Capa opcional para la lógica de negocio pura.
